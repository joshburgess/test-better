# Public API snapshots

Each `<crate>.txt` file in this directory is a snapshot of one workspace
crate's public API, generated by [`cargo-public-api`]. It lists every public
item (types, functions, traits, impls, re-exports) reachable from that crate's
root, with `--all-features` enabled so optional surface is included too.

These snapshots exist so a change to the public API is always visible in
review: a diff to one of these files is a diff to what downstream users can
see. The `public-api` CI job (and `scripts/check-public-api.sh`) regenerates
the live surface and fails if it has drifted from what is committed here.

## Regenerating

When a public API change is intentional, regenerate and commit the snapshots:

```sh
scripts/check-public-api.sh --write
```

This needs `cargo-public-api` (`cargo install cargo-public-api`) and a
`nightly` toolchain available to rustup; `cargo-public-api` drives nightly
rustdoc to produce the JSON it reads. The snapshots themselves are
toolchain-stable text, so they are reviewed and committed like any other file.

[`cargo-public-api`]: https://github.com/Enselic/cargo-public-api
